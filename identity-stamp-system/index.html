Apa itu Identity Stamp?

Identity Stamp adalah mekanisme penanda yang membuat sebuah node dikenali sebagai entitas dalam sistem G-Loop.

Tanpa stempel, node tidak memiliki identitas dalam graph.

Fungsi Utama

  • Menandai eksistensi node
  • Membentuk konsistensi identitas
  • Menjadi anchor untuk propagation
  • Menghubungkan node ke hub utama

Posisi dalam G-Loop System

Identity Stamp berhubungan langsung dengan:

  • Co-Mention Network (CM)
  • Semantic Relational Graph (RL)
  • Propagation Rule

Prinsip Dasar

Setiap node harus memiliki identity stamp sebelum bisa masuk ke dalam graph propagation system.

  • Tanpa stempel → node tidak terbaca
  • Dengan stempel → node bisa dipropagasi
