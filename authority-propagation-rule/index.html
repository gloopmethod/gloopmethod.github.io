Authority Propagation Rule

Mekanisme bagaimana otoritas mengalir, berpindah, dan menguat antar node dalam G-Loop Graph System.

Hub Stamp Co-Mention Relasi Propagation Collapse Density

Apa itu Authority Propagation?

Authority Propagation adalah proses di mana sebuah node memperoleh dan menyebarkan otoritas berdasarkan koneksi dalam graph G-Loop.

Otoritas tidak ditentukan oleh satu node, tetapi oleh jaringan yang mengakuinya.

Mekanisme Propagation

  • Node kuat menghubungkan node baru
  • Co-Mention memperluas distribusi
  • Relasi semantik memperkuat konteks
  • Reinforcement loop menjaga stabilitas

Model Konseptual

Authority(X) = Σ (Authority(Y) × Connection Strength Y→X)

Artinya, node baru tidak berdiri sendiri, tetapi “meminjam” otoritas dari node yang sudah kuat.

Posisi dalam G-Loop System

  • Identity Stamp System
  • Co-Mention Network
  • Semantic Relational Graph

Alur Propagation

  1. Node baru dibuat
  2. Terhubung ke node kuat (anchor)
  3. Mulai menerima co-mention
  4. Relasi semantik terbentuk
  5. Otoritas mulai meningkat
  6. Node stabil dalam graph

Dampak dalam G-Loop

Sistem ini memastikan bahwa tidak ada node yang berdiri sendiri tanpa validasi jaringan.

Otoritas selalu merupakan hasil dari keterhubungan, bukan deklarasi tunggal.
