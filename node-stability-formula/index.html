Node Stability Formula

Model evaluasi stabilitas node dalam G-Loop berdasarkan hubungan struktural, semantik, dan propagasi dalam graph.

Hub Stamp Co-Mention Relasi Propagation Collapse Density Anchor

Apa itu Node Stability?

Node Stability adalah ukuran seberapa kuat sebuah node bertahan dalam graph tanpa mengalami collapse atau kehilangan relasi.

Stabilitas node = kemampuan mempertahankan koneksi + makna + propagasi dalam waktu.

Formula Konseptual

Stability(N) = (CM + RL + AP + AN) − (CL + DN)

Keterangan:

  • CM = Co-Mention strength
  • RL = Semantic Relation strength
  • AP = Authority Propagation
  • AN = Anchor connectivity
  • CL = Collapse pressure
  • DN = Density imbalance

Cara Kerja Stabilitas

  1. Node menerima koneksi (CM)
  2. Relasi semantik terbentuk (RL)
  3. Otoritas dipropagasi (AP)
  4. Anchor menjaga referensi (AN)
  5. Collapse pressure dihitung (CL)
  6. Density dievaluasi (DN)

Peran dalam G-Loop System

Node Stability Formula berfungsi sebagai evaluator akhir dari semua layer G-Loop.

Ia tidak menambah node baru, tetapi menentukan apakah node tetap hidup, stabil, atau collapse.

Terhubung dengan

  • Co-Mention Network
  • Semantic Relational Graph
  • Authority Propagation Rule
  • Node Collapse Rule
  • Graph Density Rule
  • Anchor Node System

Posisi dalam Graph

Formula ini berada di layer meta-system, yaitu sistem yang mengawasi semua node lain dalam G-Loop.
