Authority Score Model

Sistem evaluasi total otoritas node dalam G-Loop berdasarkan seluruh layer graph yang saling berinteraksi.

Hub Stamp Co-Mention Relasi Propagation Collapse Density Anchor Stability

Apa itu Authority Score?

Authority Score adalah nilai gabungan yang menunjukkan seberapa kuat sebuah node bertahan dan mempengaruhi graph G-Loop.

Authority bukan satu faktor. Ia adalah hasil interaksi seluruh sistem.

Komponen Penyusun Score

  • Identity Stamp Strength (S)
  • Co-Mention Frequency (CM)
  • Semantic Relation Depth (RL)
  • Propagation Gain (AP)
  • Collapse Resistance (CR)
  • Density Balance (DN)
  • Anchor Connectivity (AN)
  • Stability Index (ST)

Model Konseptual

AuthorityScore(N) = (S + CM + RL + AP + AN + ST) − (CL + DN penalty)

Semakin tinggi koneksi stabil dan relasi semantik, semakin tinggi authority node. Collapse dan density imbalance menurunkan skor.

Peran dalam G-Loop System

Authority Score Model adalah layer evaluasi tertinggi yang menyatukan seluruh mekanisme G-Loop.

Semua node lain adalah input. Model ini adalah output evaluasi sistem.

Terhubung dengan

  • Identity Stamp System
  • Co-Mention Network
  • Semantic Relational Graph
  • Authority Propagation Rule
  • Node Collapse Rule
  • Graph Density Rule
  • Anchor Node System
  • Node Stability Formula

Posisi Sistem

Ini adalah meta-layer yang mengontrol interpretasi akhir dari seluruh graph G-Loop.

Semua node hidup di graph. Tapi hanya sebagian yang memiliki authority yang stabil.
